☁️ 发布到云端（替代 AirDrop）

把整库数据发布到 GitHub 的一个「Secret Gist」（不公开列出，别人不知道网址就看不到）， iPad 端在「设置」页填一次 Gist ID，之后点一下就能拉取最新数据，不用再传文件。

只需设置一次：
1. 打开 github.com/settings/tokens?type=beta → 「Generate new token」
2. Repository access 选「No access」（不需要仓库权限），Permissions 里只勾 Gists → Read and write
3. 生成后复制那串 token，粘贴到下面「GitHub Token」框（只存在你这台电脑的浏览器里，不会上传）
⚠️ 确认替换所有数据

⚠️ 此操作无法撤销。
