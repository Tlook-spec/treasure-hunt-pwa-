## iPad PWA 主屏幕图标 + 飞行模式（关键）

**⚠️ 必须按顺序操作，否则飞行模式会打不开未缓存的子页面**

**准备阶段（有网）**
- [ ] 从主屏幕图标启动 App（如无图标，先 Safari 访问 → 「添加到主屏幕」）
- [ ] 依次点进每个页面：首页 → 开始游戏（进入探险选择页）→ 返回 → 设置（进入设置页）→ 返回
- [ ] 等约 3 秒，让 Service Worker 动态缓存所有已访问的资源
- [ ] Safari F12 / Eruda Console 确认无红色报错

**飞行模式阶段**
- [ ] 切换 iPad 到「飞行模式」
- [ ] 从主屏幕图标重新启动 App → 首页正常显示（SW 缓存有效）

- [ ] 点「⚙️ 设置」→ 设置页正常显示
