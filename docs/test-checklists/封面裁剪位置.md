# 功能名：编辑端封面图裁剪位置（横向/纵向滑块 + 实时预览）

> L1 封面图在孩子端会被 `object-fit:cover` 裁剪（列表方块 60×60、开始页横幅 ~16:9），可能裁到半张脸。
> admin 封面区新增：上传后显示**两个裁剪预览**（横幅 + 方块）+ **横向/纵向滑块**，存 `Level.coverPosition`（object-position），
> play 列表缩略图 / 开始页横幅套用。默认 "50% 50%"（居中）。

---

## 桌面浏览器测试（Windows + Chrome，admin）
- [ ] 编辑 L1 → 上传一张人脸偏上的封面 → 出现「开始页横幅」+「列表缩略图」两个预览 + 横向/纵向滑块
