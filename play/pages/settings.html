📱 怎么把 JSON 文件传到 iPad？
1
AirDrop（最快）：Mac 导出 JSON → AirDrop 到 iPad → iPad「文件」App → 分享 → 用 Safari 打开
2
邮件 / 微信：发给自己 → iPad 上打开附件 → 用 Safari 打开
3
iCloud Drive：把 JSON 放进 iCloud → iPad「文件」App 找到 → 分享 → 用 Safari 打开
