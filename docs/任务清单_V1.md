# 寻宝游戏 PWA - 任务清单(V1,v1.9)

> 这是 MVP 之后的 V1 作战手册。**对应 PRD v1.9 + CLAUDE.md v1.9**。
>
> **V1 主题**:先把单人体验做香(命中真机反馈)→ 再做探险地图+故事+海报(沉浸感)→ 收尾杂项 → 最后才考虑多人(最不确定,玩过前面再定)。
>
> **编码规则**(同 MVP 的 T/M 流水码):全 V1 用 **V1-01、V1-02 … 一路连续往下排**,分区(A/B/C/D/E)只是标题,不重新计数。
>
> **使用方法**(同 MVP):
> 1. 严格按顺序做,前置没完成不要跳(MVP 串行推进原则继续生效)
> 2. 每个任务的【复制给 Claude Code 的指令】整段复制进 Claude Code
> 3. 完成后用【验证清单】自查,在 `[ ]` 里打钩 `[x]`
> 4. 每做完 1 个任务 → 单行英文 git commit + push → 更新 PROGRESS.md
> 5. 游戏端每块做完必测 **iPad 三态**(Safari / PWA 有网 / PWA 飞行模式)
>
> **模型**:日常用 Sonnet;遇到难题或多文件大改动,`/model` 切 Opus,解决后切回。
