# 功能名：V1-02 通关成绩单 + 错题本

## 桌面浏览器测试（Windows + Chrome）

- [ ] 打一局：多站中故意制造以下三种情况：
  - 站 A：**一道错满 3 次保底**（score=0）+ 其他题正常
  - 站 B：**一道错 2 次但第 3 次自己答对**（score=5）+ 其他题正常
  - 站 C：**全部题一次答对**
- [ ] 通关页出现「📋 按站成绩」区域，每站各一张卡片
- [ ] **站得分正确**：每站得分 = 该站 questionResults 的 score 之和
- [ ] **「共 Y 题，一次答对 X 题」正确**（X = answerAttempts===1 的题数）
- [ ] **保底那道（score=0）**：标注「这题没答出来，直接看了答案」；显示「❌ 选了：X. 选项文字」；显示「✅ 正确答案：X. 选项文字」；有解析时显示「📖 解析：...」
- [ ] **错 2 次后自己答对那道（score=5）**：列在错题区；**不标**「这题没答出来」← 重点验证
- [ ] **全对站（站 C）**：显示「这站全对，厉害！🌟」，不列错题
- [ ] 没填解析的错题 → 不出现空的「📖」行
- [ ] 选错了但 wrongChoices 为空的旧存档 → 显示「❌ 选错了（未记录具体选项）」降级文案，不报错
- [ ] 有错题的站默认展开；全对的站默认折叠
- [ ] 点击全对站的标题栏 → 展开/折叠正常
- [ ] 刷新通关页 → 成绩单仍在、用时不变（验结算幂等，endedAt 不被覆盖）
- [ ] F12 Console 无红色错误
