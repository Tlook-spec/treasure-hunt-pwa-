# 功能名：V1-27 游戏端开局多人创建

> 在 `start-level.html`（「开始冒险」页）加入多人玩家创建流程，插在「开场故事」之前。
> 选人数 1 = 单人，流程和以前完全一样（零改动）。选 2~6 人，点「开始冒险」先进玩家创建视图，
> 每人填名字/编号/颜色/可选头像，确认后保存 players 到 GameSession、头像存 photos 表。
