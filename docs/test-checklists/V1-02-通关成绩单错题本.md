# 功能名：V1-02 通关成绩单 + 错题本

## 桌面浏览器测试（Windows + Chrome）

> 统一口径：答对 = 最终得分 > 0；错题 = 最终得分 === 0（没答出来、保底）。错题本只收 score===0 的题。

- [ ] 打一局：多站中故意制造以下三种情况：
  - 站 A：**一道错满 3 次保底**（score=0）+ 其他题正常
  - 站 B：**一道错 2 次但第 3 次自己答对**（score=5）+ 其他题正常
  - 站 C：**全部题答对**（每题最终都得分>0）
- [ ] 通关页出现「📋 按站成绩」区域，每站各一张卡片
- [ ] 折叠副行显示「X 分 · 答对 X/Y 道题 · 点击查看详情吧」；**答对数 = score>0 的题数**（验证：得 15 分时答对数不为 0）
- [ ] **站得分正确**：每站得分 = 该站 questionResults 的 score 之和
- [ ] 展开后「共 Y 题，答对 X 题」正确（X = score>0 的题数）
- [ ] **保底那道（score=0）**：进错题本，标注「这题没答出来，直接看了答案」；显示「❌ 选了：X. 选项文字」；显示「✅ 正确答案：X. 选项文字」；有解析时显示「📖 解析：...」
- [ ] **错 2 次后自己答对那道（score=5）**：算答对，**不进错题本** ← 重点验证
- [ ] **全对站（站 C，无 score===0 题）**：显示「这站全对，厉害！🌟」，不列错题
- [ ] 没填解析的错题 → 不出现空的「📖」行
- [ ] 选错了但 wrongChoices 为空的旧存档 → 显示「❌ 选错了（未记录具体选项）」降级文案，不报错
- [ ] 所有站默认折叠
- [ ] 点击任意站标题栏 → 展开/折叠正常，副行去掉「点击查看详情吧」
- [ ] 刷新通关页 → 成绩单仍在、用时不变（验结算幂等，endedAt 不被覆盖）
- [ ] F12 Console 无红色错误
