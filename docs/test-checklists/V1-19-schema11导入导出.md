## 桌面浏览器测试（Windows + Chrome，编辑端）

### 导出验证：新文件写 "1.1" + 地图故事字段包含

前提：有一个带地图底图 + 点位坐标 + 三段文案 + 开场/结局故事的 L1

- [ ] 编辑端「导入导出」→ 点「导出」，下载 JSON 文件
- [ ] 用文本编辑器打开 JSON，确认第一行是 `"schemaVersion": "1.1"`
- [ ] 确认 `data.levels[0]` 包含字段：`mapImage`（base64 字符串）、`mapFontSize`、`mapNameColor`、`mapNameColorCompleted`、`openingStory`、`endingStory`
- [ ] 确认 `data.points[0]` 包含字段：`mapX`、`mapY`、`questionIntroText`、`completionText`
- [ ] F12 Console 无红色错误

- [ ] 打开游戏端「导入」页，导入上面导出的 1.1 文件
