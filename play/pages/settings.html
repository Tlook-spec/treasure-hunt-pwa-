📱 怎么把电脑上的 JSON 文件传到 iPad？
1
AirDrop（最方便）：Mac 导出 JSON → 直接 AirDrop 到 iPad → iPad「文件」App 里找到 → 点击分享 → 用 Safari 打开
2
邮件 / 微信：发给自己 → iPad 上打开附件 → 用 Safari 打开
3
iCloud Drive：把 JSON 放进 iCloud → iPad「文件」App 找到 → 点击分享 → 用 Safari 打开
