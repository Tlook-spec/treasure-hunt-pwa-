# 功能名：M12 打印应急小抄

## 前置条件

- 编辑端已有至少 1 个探险（L1），其中有 3-5 个已保存的点位（L2）
- 部分点位填有「家长备忘」，部分留空（验证「—」占位）

---

## 桌面 Chrome 测试 — 页面加载

- [ ] 在 L1 详情页点「📋 打印应急小抄」→ **新标签页**打开小抄页
- [ ] 页面顶部显示 iPad 打印警告横幅
- [ ] 标题显示 L1 名称（大字）
- [ ] 副标题显示「应急小抄 · 共 N 个点位」
- [ ] 蓝色说明横幅文字正确显示
- [ ] 表格包含所有点位，行数与点位数一致
- [ ] 表格列：序号 / 点位名称 / 数字码 / 家长备忘
- [ ] 数字码用等宽字体、加粗、字间距宽
- [ ] 有备忘的行显示备忘内容，无备忘显示「—」
- [ ] 底部显示打印日期和「寻宝游戏」字样
- [ ] F12 Console 无红色错误

## 桌面 Chrome 测试 — 打印

- [ ] 点「🖨️ 立即打印」→ 打印对话框出现
- [ ] 打印预览中：工具栏和 iPad 警告**不出现**
- [ ] 打印预览中：A4 一页能容纳 5-10 行（不分页）
- [ ] 数字码字体清晰可读（15px+、等宽、加粗）
- [ ] 表头蓝色背景在打印预览中可见（或退为黑白也可接受）
- [ ] 「← 返回」按钮 → 回到上一页

## 桌面 Chrome「另存为 PDF」

- [ ] 选「另存为 PDF」→ 生成 PDF
- [ ] PDF 排版：A4 一页，表格完整
- [ ] PDF 中数字码清晰可读，没有截断
- [ ] PDF 底部打印日期可见

## iPad Safari 测试

- [ ] iPad 打开页面，警告横幅正常显示
- [ ] 表格正常渲染，数字码清晰
- [ ] 点「立即打印」→ iPad 打印对话框出现
- [ ] 选「另存为 PDF」→ PDF 排版可接受（允许轻微差异）

## 异常情况

- [ ] 直接访问 `print-cheatsheet.html`（不带 levelId）→ 友好错误提示
- [ ] 传入不存在的 levelId → 友好错误提示
- [ ] 探险下无点位 → 显示「还没有点位」提示
