# 功能名：V1-18 成就海报

> victory.html 通关庆典页新增「🎁 生成成就海报」按钮，
> 点击后 Canvas 合成 1080px 竖版海报，孩子长按可保存到相册。
> 包含：蓝色渐变头部（探险名/总分/用时）、站点列表、16:9 合影网格、全亮地图。

---

## 桌面浏览器测试（Windows + Chrome，Live Server）

### 前提：完整通关一局（至少 3 站，有照片，有地图）

**按钮与状态**

- [ ] 通关庆典页底部出现「🎁 生成成就海报」按钮
- [ ] 点击按钮 → 按钮变灰禁用 + 显示「⏳ 生成中…」
- [ ] 1-3 秒内海报图片出现，状态文字消失
- [ ] 按钮文字变为「🎁 重新生成海报」并恢复可点击
- [ ] 再次点击可重新生成（不报错）

**海报内容验证**

- [ ] 头部：探险名称显示正确
- [ ] 头部：总分数字醒目大字
- [ ] 头部：用时格式正确（如「12 分 34 秒」）
- [ ] 站点列表：各站名 + 得分 + 是否用过求助，行数与实际点位数一致
- [ ] 合影网格：显示拍过的照片，16:9 比例裁切，圆角
- [ ] 地图（有地图时）：底图可见，所有点位显示红五角星 + 金色站名
- [ ] 地图上星星数量等于点位数量

**点击交互**

- [ ] 长按/右键图片可弹出保存菜单（Chrome 上右键「图片另存为」可用）
- [ ] 点击图片 → 全屏查看（复用 openFullscreen 逻辑）
