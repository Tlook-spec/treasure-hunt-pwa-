# 功能名：V1-02 通关成绩单 + 错题本

## 桌面浏览器测试（Windows + Chrome）

> 统一口径：答对 = 最终得分 > 0；错题 = 最终得分 === 0（没答出来、保底）。
> 两级折叠：站卡（一级）展开后逐题列出**所有题目+各自得分**；每道题（二级）可再展开看详情。

- [ ] 打一局：多站中故意制造以下三种情况：
  - 站 A：**一道错满 3 次保底**（score=0）+ 其他题正常
  - 站 B：**一道错 2 次但第 3 次自己答对**（score=5）+ 其他题正常
  - 站 C：**全部题答对**（每题最终都得分>0）
- [ ] 通关页出现「📋 按站成绩」区域，每站各一张卡片
- [ ] **所有站卡默认折叠**；折叠副行显示「X 分 · 答对 X/Y 道题 · 点击查看详情吧」；答对数 = score>0 的题数（验证：得 15 分时答对数不为 0）
- [ ] **站得分正确**：每站得分 = 该站 questionResults 的 score 之和
- [ ] 点击站卡展开 → 副行去掉「点击查看详情吧」；展开后「共 Y 题，答对 X 题」正确
- [ ] **站卡展开后逐题列出所有题目**（不只错题），每行显示题干 + 该题得分
- [ ] **0 分题**：得分数字标红、整行淡红底
- [ ] **二级展开（点题目行）**：显示选错答案（如有）/ 正确答案「X. 选项文字」/（有则）解析
- [ ] **0 分题二级默认展开**，并标「这题没答出来，直接看了答案」
- [ ] **错 2 次后自己答对那道（score=5）**：得分正常色、二级默认折叠；点开后能看到「❌ 选错：...」 ← 重点验证
- [ ] **一次答对的题（score=10）**：二级展开无「❌ 选错」行（wrongChoices 为空），只有正确答案/解析
- [ ] **全对站（站 C）**：展开顶部有「这站全对，厉害！🌟」，仍逐题列出
- [ ] 没填解析的题 → 二级展开不出现空的「📖」行
- [ ] 旧存档（无 wrongChoices）→ 不报错，错题二级展开不显示「❌ 选错」行（降级）
- [ ] 刷新通关页 → 成绩单仍在、用时不变（验结算幂等，endedAt 不被覆盖）
- [ ] F12 Console 无红色错误
