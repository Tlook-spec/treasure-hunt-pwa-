## 桌面浏览器测试（Windows + Chrome）

### 三段文案都填（发现文案 + 地图（有底图） + 答题引导）

- [ ] 扫码/输码成功 → 进 quiz.html → 先看到发现文案页（大字 + 「继续 →」按钮）
- [ ] 点「继续 →」→ 全屏地图覆盖层弹出，当前点位出现白色空心圆 + 名字（浮现淡入约 0.5s）
- [ ] 前面已完成的点位显示为红色五角星印章 + 金色名字
- [ ] 地图约 1 秒后自动关闭 → 弹出答题引导弹窗（大字 + 「开始答题！」按钮）
- [ ] 点「开始答题！」→ 正式进入题目页面
- [ ] 答题过程不受影响（判分、求助、保底通关逻辑均正常）

### 只填发现文案（无答题引导）

- [ ] 发现文案页 → 点「继续 →」→ 地图浮现（若有底图）→ 直接进题目（不弹引导弹窗）

### 只填答题引导（无发现文案）

- [ ] 直接弹出地图（若有底图）→ 地图关闭 → 答题引导弹窗 → 「开始答题！」→ 题目

- [ ] 有底图：直接弹出地图浮现 → 地图关闭 → 直接进题目
- [ ] 无底图：直接进题目（无任何弹窗）
