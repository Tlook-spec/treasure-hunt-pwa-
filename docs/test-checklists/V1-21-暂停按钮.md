# 功能名：V1-21 主动暂停按钮

> 答题屏(quiz.html)+ 找站屏(hint.html)顶部蓝条最左端加 ⏸ 暂停按钮。
> 点击 → 全屏遮罩盖住屏幕 + 「▶ 继续游戏」。暂停期间计时不涨：
> pauseStartedAt/pausedMs 存 session(不进导出)，继续时累加 pausedMs，通关用时扣掉它。

---

## 桌面浏览器测试（Windows + Chrome，Live Server）

### 暂停 / 继续基本流程

- [ ] 进入答题屏 → 顶部蓝条最左端有 ⏸ 按钮（在求助 💡 左边、不挡题号）
- [ ] 点 ⏸ → 全屏深色遮罩：「⏸ 已暂停 / 休息一下…」+「▶ 继续游戏」
- [ ] 遮罩盖住时，下面的选项点不到（被挡住）
- [ ] 点「继续游戏」→ 遮罩消失，回到原题，题号/选项/求助次数都不变
- [ ] hint 找站屏：顶部条最左端也有 ⏸ → 点击同样遮挡 + 继续

### 计时扣除（核心）

- [ ] 开一局，答到一半点 ⏸ → 等约 30 秒 → 点继续 → 打完通关
- [ ] 通关页「用时」**没有**把那 30 秒算进去（和不暂停时接近）
- [ ] F12 看 session：`pausedMs` 约等于暂停的毫秒数，`pauseStartedAt` 继续后变回 null

### 暂停中关闭再续玩（关键）

- [ ] 答题屏点 ⏸ 进入遮罩 → 直接关掉标签页/Live Server（模拟关 App）
- [ ] 重新打开，从「继续上次探险」进入 → **直接回到暂停遮罩**（没有偷偷计时）
- [ ] 点继续 → 正常答题 → 通关用时把「关闭期间」也算进暂停、没计入用时

### 向后兼容 / 不破坏

- [ ] 老存档（没有 pausedMs/pauseStartedAt 的旧 session）→ 进游戏、通关用时正常，无报错
- [ ] 求助按钮、🗺️ 地图 FAB、撒花、结算用时幂等（刷新通关页用时不变）都没被影响
- [ ] F12 Console 全程无红色错误
