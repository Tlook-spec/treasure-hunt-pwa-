# 手动测试 Checklist：L3 题库 CRUD（M09）

## 桌面浏览器测试（Windows + Chrome）

### 列表页基础
- [ ] 进入「题库管理」→ 显示「题库还没有题目，点击新建题目添加第一道！」
- [ ] 顶部有「➕ 新建题目」「📥 CSV 导入」「📤 题库CSV导出」三个按钮
- [ ] 筛选条显示：全部学科下拉、难度按钮组、全部年龄段下拉、使用次数按钮组、搜索框
- [ ] F12 Console 无红色错误

### 新建题目（选择题）
- [ ] 点「新建题目」→ 切换到表单面板，标题「新建题目」
- [ ] 默认题型为「选择题」（选项高亮蓝色）
- [ ] 默认有 A、B 两个选项输入框
- [ ] 「添加选项」按钮可见，点击 → 新增 C 选项；再点 → 新增 D 选项；D 之后按钮消失
- [ ] 删除 C 选项 → 选项从 A/B/C/D 变回 A/B/C（字母重新排列）
- [ ] 填入题干、A/B 两个选项、点正确答案 A（圆圈变绿）→ 选学科「数学」→ 点「保存题目」
- [ ] 保存后回到列表，出现新题目行（显示题干前 30 字 + 「选择」「数学」标签 + 🔵 0次）
- [ ] 列表计数「共 1 道题目」

### 新建题目（判断题）
- [ ] 点「新建题目」→ 点「判断题」按钮 → 选项区自动变为「对」「错」两行（只读，不可编辑）
- [ ] 「添加选项」按钮消失
- [ ] 正确答案区只有 A、B 两个圆圈（A=对，B=错）
- [ ] 填题干、选 B、选学科 → 保存 → 列表出现「判断」标签的新题

### 必填校验
- [ ] 不填题干 → 保存 → alert「请填写题目内容」
- [ ] 不选学科 → 保存 → alert「请选择学科」
- [ ] 选项输入框留空 → 保存 → alert「请填写所有选项内容」
- [ ] 不选正确答案 → 保存 → alert「请选择正确答案」

### 编辑题目
- [ ] 点某题行「✏️ 编辑」→ 表单面板打开，标题「编辑题目」，所有字段已填入当前值
- [ ] 题型、选项、正确答案均正确回显
- [ ] 修改题干后保存 → 列表更新

### 复制题目
- [ ] 点「📋 复制」→ 列表末尾新增一行，题干相同（不含「副本」字样，这是规格）
- [ ] 复制出的题目 usedCount = 🔵 0次（重置）

### 删除（无引用）
- [ ] 点「🗑️ 删除」→ 弹出「🗑️ 确认删除」弹窗，含题干预览和「此操作无法撤销」
- [ ] 点「取消」→ 弹窗关闭，题目不删
- [ ] 点「确认删除」→ 弹窗关闭，题目消失，列表刷新

### 删除（有引用保护）
- [ ] 先在 L2 点位里绑定这道题，再来题库删除它
- [ ] 点「🗑️ 删除」→ 弹窗标题「⚠️ 无法删除」，显示被哪些探险/点位引用
- [ ] 弹窗只有「取消」按钮，没有「确认删除」
- [ ] 关闭弹窗后题目未被删

### 筛选功能
- [ ] 多建几道题（不同学科/难度/年龄段/usedCount）
- [ ] 学科下拉选「数学」→ 只显示数学题
- [ ] 点难度「★★」→ 只显示 2 星题；再点「★★」→ 取消，恢复全部
- [ ] 年龄段下拉选「7-9」→ 只显示 7-9 岁题
- [ ] 使用次数「0次」→ 只显示 usedCount=0 的题；「2+次」→ usedCount≥2
- [ ] 搜索框输入题干关键词 → 实时过滤
- [ ] 筛选后计数显示「筛选结果：X 道（共 Y 道）」
- [ ] 无结果时显示「没有符合筛选条件的题目，调整筛选条件试试」

### CSV 导出
- [ ] 题库有几道题后点「📤 题库CSV导出」→ 浏览器下载 `.csv` 文件
- [ ] 文件名格式「题库-2025-xx-xx.csv」
- [ ] 用记事本/Excel 打开 → 中文正常显示（UTF-8 BOM），列名正确
- [ ] 题库为空时点导出 → alert「题库里还没有题目，无法导出」

### CSV 导入占位
- [ ] 点「📥 CSV 导入」→ alert「CSV 导入功能将在 M10 实现」

### Console 检查
- [ ] 全程无红色错误

## 注意事项

- **correctAnswer 格式**：存字母字符串 'A'/'B'/'C'/'D'，与游戏端答题页一致
- **usedCount** 新建和复制时均为 0；编辑保存不改变 usedCount
- **判断题**：options 固定 ['对','错']，correctAnswer 为 'A'（对）或 'B'（错）
- 难度字段可为 null（用户不选时），列表显示「—」
