# 功能名：V1-10 编辑端地图标记编辑器

> 改动范围：`admin/pages/levels.html`（工具区加按钮）、`admin/scripts/level-detail.js`（显示/隐藏按钮）、`admin/pages/map-marker.html`（新建独立页）、`admin/scripts/map-marker.js`（新建编辑器逻辑）

**前提**：先在 V1-09 里为某探险上传了地图底图

---

## 桌面浏览器测试（Windows + Chrome）

### 入口按钮

- [ ] L1 详情页：已上传地图的探险 → 工具区出现「📍 地图标记」按钮
- [ ] L1 详情页：未上传地图的探险 → 「📍 地图标记」按钮不显示
- [ ] 点击「📍 地图标记」→ 在新标签页打开地图标记编辑器，地图图片正常加载

### 放置标记（按顺序）

- [ ] 依次点击地图空白处 → 标记按点位 order 依次绑定（圆圈数字 1、2、3…）
- [ ] 右侧面板：每放一个标记，对应点位 ⬜ 变 ✅
- [ ] 右侧提示文字：「还差 X 个」随放置数量递减
- [ ] 所有点位放满后提示「所有 N 个点位已标注，可以保存了！」

### 拖动调整

- [ ] 按住某个标记圆点拖动 → 标记跟随鼠标移动，松开后停在新位置
- [ ] 拖动后右侧面板状态不变（仍为 ✅）
- [ ] 拖到地图边缘：标记被限制在 0~100% 范围内，不超出图片

### 点击标记弹菜单

- [ ] 单击某个标记圆点（不拖动）→ 弹出「🗑️ 删除此标记 / ✕ 取消」小菜单
- [ ] 选「删除此标记」→ 标记消失，右侧该点位变回 ⬜
- [ ] 选「取消」或点菜单外部 → 菜单关闭，标记不变

### 预览模式

- [ ] 勾选「游戏预览」→ 所有标记变为白色空心圆 + 下方显示点位名字
- [ ] 名字颜色与 L1 设置的「名称颜色（发现时）mapNameColor」一致
- [ ] 预览模式下无法拖拽（cursor 为 default），点击标记无上下文菜单
- [ ] 取消勾选 → 回到蓝色实心圆 + 序号编辑模式

### 清空所有

- [ ] 点「🗑️ 清空所有标记」→ 弹出浏览器 confirm 二次确认
- [ ] 确认 → 所有标记消失，右侧全变 ⬜
- [ ] 取消 → 标记保持不动

### 保存校验

- [ ] 尚有未标注点位，点「保存」→ 弹 alert 提示「还有 X 个未标，未标点位：…」，不写 DB
- [ ] 全部标注后点「保存」→ alert「✅ 坐标已保存！」
- [ ] 关闭标签页，重新打开同一探险的地图标记编辑器 → 标记位置与上次保存一致
- [ ] DevTools → IndexedDB → points 表 → 各点位 mapX/mapY 为 0~1 小数

### 兼容性：无地图

- [ ] 直接访问 `map-marker.html?levelId=xxxx`（该探险无地图）→ 弹 alert 提示，页面关闭

### F12 检查

- [ ] 整个操作过程 Console 无红色错误
