## 🤝 给 Claude Code 的使用提示

开发任意 M 任务前，请先：

1. **打开 PRD 对应章节**（见上表的"PRD 章节"列）
2. **看对应的线框图**（见"文件名"列）
3. 再开始写代码

**示例话术**（你可以直接复制给 Claude Code）：

```
我要开始 M07 L1 探险 CRUD。请先：
1. 看 docs/wireframes/admin/01-l1-list.png 和 02-l1-form-and-detail.png
2. 读 docs/PRD.md §4.1.2
3. 完成后告诉我准备怎么实现，我确认后再动手
```

## 🛠️ 维护流程

### 场景 1：增加一张新线框图

1. 在 `source.excalidraw` 里画
2. 框选 → 导出 PNG 到 `admin/` 或 `play/`，文件名按规则
3. **更新本 README**：在对应表格加一行
4. 如果同时调整了任务范围 → 也改 `任务清单_MVP_v1.7.md` 里的 T10

### 场景 2：删除/合并一张线框图

1. 删 PNG 文件（源文件里那部分可以保留作为参考）
2. **更新本 README**：从主表格移除，加到"已删除/合并"清单里
3. 在下方"更新记录"加一行
4. 如果原本是 T10 任务清单里的必画项 → 改任务清单

### 场景 3：改了某张图的内容

1. 打开 `source.excalidraw` 改
2. 重新导出 PNG 覆盖
3. 如果**只改样式**（按钮颜色、间距）→ 不用改 README
4. 如果**改了交互流程**（多了按钮、改了跳转）→ 在下方"更新记录"加一行

### 不需要改的文档

- ❌ `PRD.md` — PRD 写功能，线框图写表现，互不干涉
- ❌ `CLAUDE.md` — 除非影响协作约定（基本不会）
- ❌ `pages-data.md`（T11 任务的产物）— 那是数据清单，跟线框图独立
