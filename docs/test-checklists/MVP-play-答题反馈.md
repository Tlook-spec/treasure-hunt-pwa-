## 桌面浏览器测试（Windows + Chrome）

### 答题判定

- [ ] 选择正确选项 → 该选项变绿 → 出现橙色「🎉 答对啦！」大卡 → 约 1 秒后自动切到下一题
- [ ] 选择错误选项（第 1 次）→ 该选项变红 + 抖动动画 → 出现「答案不正确」深色卡（含题目 hint） → 0.6 秒后卡片消失，四个选项恢复可点
- [ ] 选择错误选项（第 2 次）→ 同上，再次可重选
- [ ] 选择错误选项（第 3 次）→ 该选项变红，**正确选项变金色高亮** → 出现「没关系，下次记住啦！」深色卡 → 约 1.5 秒后自动切到下一题
- [ ] 当前站点最后一道题答对（或第 3 次错）→ 跳转到 `photo.html`（M24 前 404 属正常）

### 计分逻辑

- [ ] 一次答对，不用求助 → 后台记录 score=10（可在 settings.html 导入前翻 IndexedDB 验证）
- [ ] 答错 1-2 次后答对，不用求助 → score=5
- [ ] 用了求助后答对 → score=3
- [ ] 第 3 次错保底 → score=0

### 求助按钮

- [ ] 横幅右侧「💡 求助 3」按钮可见
- [ ] 点击求助 → 选项禁用 → 出现蓝色「确定求助爸爸妈妈吗？」卡，有「确定」「不用了」两个按钮
- [ ] 点「确定」→ 扣除次数（横幅数字 -1）→ 显示「快去问爸爸妈妈！」→ 2 秒后卡片消失，选项恢复可点
- [ ] 点「不用了」→ 卡片立即消失，选项恢复
- [ ] 求助次数耗尽后再点 → 出现「已经没有求助次数咯」蓝色卡 → 2 秒后自动消失
- [ ] `usedHelpThisQuestion=true` 后答对 → score=3（即使0次答错）

### 防重复点击

- [ ] 答题选项点击后立即禁用，不允许在动画/延迟期间重复点击

### F12 Console

- [ ] 无红色错误
- [ ] 无「题目索引超出范围」「游戏会话不存在」等异常日志
