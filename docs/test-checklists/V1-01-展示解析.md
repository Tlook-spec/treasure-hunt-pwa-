# 功能名：V1-01 保底通过时展示解析

## 桌面浏览器测试（Windows + Chrome）

- [ ] 同一题故意答错 3 次（保底），题目**已填写解析** → 反馈区依次出现：「没关系，下次记住啦！🌟」→「✅ 正确答案：XXX」（纯选项文字，无 A/B 标签黏连）→「📖 解析：XXX」→「好的，记住了！」按钮
- [ ] 同一题答错 3 次（保底），题目**未填解析**（explanation 为空/null）→ 只出现正确答案行，不出现「📖」行，不出现空行
- [ ] 「正确答案」文字与选项按钮高亮（option-bust-correct）显示的文字一致，且无「A」「B」之类标签前缀
- [ ] 一次答对 → 不出现解析，1 秒后自动进入下一题，正常流程不受影响
- [ ] 用求助后答对 → 不出现解析，自动进入下一题，正常流程不受影响
- [ ] 答错第 1 次 → 只出现「答案不正确，再思考一下吧」+ 提示（若有），0.6 秒后恢复选项，无解析
- [ ] 答错第 2 次 → 同上，无解析
- [ ] 点击「好的，记住了！」→ 进入下一题（或本站结束跳拍照页），流程正常
- [ ] F12 Console 无红色错误

## iPad Safari 测试

- [ ] 同一题答错 3 次（保底，题目有解析）→ 显示正确答案 + 解析行，布局不溢出屏幕
- [ ] 同一题答错 3 次（保底，题目无解析）→ 只显示正确答案，无空行
- [ ] 一次答对 → 不出现解析，自动下一题
- [ ] 用求助后答对 → 不出现解析，自动下一题
- [ ] Eruda Console（?debug=1）无红色错误
